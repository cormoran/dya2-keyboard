ご自身で発注いただくことで、ボトムケースのエンブレムプレートを真鍮に変えることができます。
重量が増加してタイピングの安定性が上がるかもしれません。あとかっこいい。

Tadpole ピンには複数の硬さがあります。ご自身で入手いただくことでタイピングの硬さを調整することができます。

一部の硬さについては booth で販売しています。

透明なケース　ご自身で発注いただくことで作っていただけますが、ねじ止め部分にeサートをつけることが推奨されるので少し難易度が高いです。
