DYA2 にはいくつかのカスタマイズオプションがあります。

## 真鍮製エンブレムプレート

ボトムケースのエンブレムプレートを 3D プリント製から真鍮製に変更できます。真鍮プレートにすることで重量が増し、タイピング時の安定性が向上します。

真鍮プレートはご自身で JLCPCB 等の基板製造サービスに発注していただく形になります。データは [GitHub リポジトリ](https://github.com/cormoran/dya2-keyboard) で公開予定です。

## Tadpole ピンの硬さ調整

Tadpole ピンには複数の硬さのバリエーションがあります。ご自身で入手することで、スイッチの底打ち感（タイピングの硬さ）を好みに合わせて調整できます。

一部の硬さは [Booth](https://cormoran707.booth.pm) でも販売しています。

## 透明ケース

ケースデータをご自身で発注することで、透明ケースを作ることができます。ただし、透明素材はねじ穴が弱くなりやすいため、ねじ止め部分に e-サートを取り付けることが推奨されます。そのため、通常のケースより難易度が上がります。
