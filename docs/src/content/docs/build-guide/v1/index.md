このページは [Booth](https://cormoran707.booth.pm/items/7627440) で配布中の DYA2 rev1.1 向けの組み立てガイドです。

DYA2 は基本的にははんだ付けなしで組み立てられるように設計されていますが、一部はんだ付け推奨箇所があります。

組み立ては大きく以下の３つのステップで進めます。

1. **[事前準備](/dya2-keyboard/build-guide/v1/preparation)** - 必要な部品と工具の確認
1. **[ファームウェア](/dya2-keyboard/build-guide/v1/firmware)** - XIAO へのファームウェア書き込み
1. **[組み立て](/dya2-keyboard/build-guide/v1/assembly)** - 基板とケースの組み立て

## 困ったときは

- 部品が不足している場合は、購入プラットフォームのメッセージ機能でお早めにご連絡ください
- 組み立てに失敗した場合や、うまく動かない場合も可能な限りサポートしますのでお気軽にご相談ください

サポートの連絡先：
- [Booth メッセージ](https://cormoran707.booth.pm/items/7627440)
- [GitHub Issues](https://github.com/cormoran/dya2-keyboard/issues)

## 免責事項

- 3Dプリントケースの印刷品質は保証していません。できるだけ綺麗なものを選別していますが、一部荒れた面があったり、バリや傷が残っていることがあります
- タッチセンサー端子の耐久性は保証していません。高頻度にタイピングされる場合はすぐに断線してしまう可能性があります
