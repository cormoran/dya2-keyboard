## 別途購入が必要な部品

キット以外に、以下の部品をご自身で準備していただく必要があります。

1. XIAO nrf52840 **plus** x2
   - 姉妹品の XIAO nrf52840 は使用できません
   - [TALPKEYBOARD](https://shop.talpkeyboard.com/products/seeed-studio-xiao-nrf52840-xiao-ble) や [勇者工房](https://shop.yushakobo.jp/products/10946) などで購入できます
1. Cherry MX 互換キースイッチ７２個
   - Kailh Choc low-profile スイッチは使えません
   - マウス操作に使うキー、特に左クリック（Mキー）にはクリック感のあるスピードスイッチがおすすめです
1. キーキャップ一式
   - 6,7, y, g, h, b に相当するキーが左右両方にあります
   - [YUZU keycaps](https://yuzukeycaps.com/c/333f67eb-9f7d-4774-9740-c19d98eecc7f) でオリジナルデザインを公開しています。キーのサイズ等の確認にもお使いください
   - 最下段(R4)は、
     - 標準的な英語配列の場合、左 `1.25u x3, 2.75U x1, 1u x1`, 右 `1U x2, トラックボール, 1Ux4` です
     - 日本語配列の場合、左 `1.25u x4, 1.5U x1, 1u x1`, 右 `1U x2, トラックボール, 1Ux4` です
       - YUZU keycaps では、1.5U はコンベックスキーが存在しない点にはご注意ください
   - R3 は右端のキーが特徴的で、shift が 1U になっています
   - その他は標準的な 60% 配列のサイズです
1. 34mm トラックボール用ボール
   - 色によっては反応が悪い場合があります
   - ペリックスの黒、ERGO M575付属の青では問題ないことを確認しています。
   - シルバーは動作が不安定でした。今後改善できた場合はケースデータを公開予定です。
1. 単４電池 x2
   - 公開しているファームウェアのデフォルト設定はニッケル水素電池の電圧特性を想定した残量計算になっています。
1. USB ケーブル x 2
   - PC 接続用に少なくとも片側が Type C のケーブルと、 左右接続用に Type C to Type C ケーブルが必要です
   - どちらも USB2.0 以上のデータ通信に対応している必要があります

## 必要な工具など

1. ニッパー
1. プラスドライバー（0 番と 1 番サイズ）
1. (オプション) はんだごて
   - 必須ではありませんが、何ヶ所かはんだづけ推奨箇所があります
