接続先は Config レイヤーの `Q` `W` `E` `R` `T` で切り替えることができます。

また、現在の接続先は右手の４つのLEDから確認することができます。

- `Q`~`R`は無線接続0番から3番に対応
- LED は一番上が0番、一番下が３番に対応

### ペアリング/接続

Config レイヤーキー（下の図の赤いキー）を押しながら `Q`~`R` を押すと対応する接続先が選択され、右側の LED が以下のどれかの状態になります

- 黄色に点滅: ペアリング待機状態 PC から BLE 接続できます
- 赤く点滅: ペアリング済みの PC に接続待ち状態
- 青く点灯: 接続完了

### ペアリング解除

ペアリングを解除するためには、Config を押しながら `Q`~`R` を５秒以上押してください。

LED が黄色く点滅しペアリング待機状態に変わります。

### BLE/USB 切り替え

USB, BLE 両方に接続した状態で Config を押しながら以下の `OUT TOG` を押すと BLE/USB どちらが優先されるかが変わります。

USB が優先されている時、BLE接続先以外の３つのLEDが緑色に点灯します。
