## 特徴

<CardGrid>
  <Card title="無線左右分割キーボード">
    自由な手の位置でタイピングできます。

    安定性を重視する場面では有線接続に切り替えることもできます。

  </Card>
  <Card title="34mm トラックボール内蔵">
    マウスを持ち替えることなくカーソル操作が可能です。
  </Card>
  <Card title="タッチセンサー搭載">
    トラックボールのスクロールモード切り替えなどに使えるタッチセンサーを左右に４つずつ搭載しています。
  </Card>
  <Card title="電池駆動" >
    左右はそれぞれ単４電池１本ずつで動作します。

    リチウムポリマーバッテリーと比べて発火の心配が低く、安心して使用できます。

  </Card>
  <Card title="Row Staggered 配列">
    使い慣れた Row Staggered 配列の 60% キーボードです。

    日本語配列にも対応しています。

  </Card>
  <Card title="自由なカスタマイズ">
    ZMKファームウェアを使っておりソースコードも公開しているので、キーマップやマクロ、細かな振る舞いを自由にカスタマイズできます。
  </Card>
</CardGrid>
