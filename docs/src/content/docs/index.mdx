## 特徴

<CardGrid>
<Card title="タッチセンサー搭載">
    キートップに触れると動作するタッチセンサーを搭載左右に４つずつ搭載できます。
    トラックボールのスクロールモード切り替えなどに便利です。
  </Card>
  <Card title="有線にも対応した無線左右分割キーボード">
    自由な手の位置でタイピングできます。

    安定性を重視する場面では、ケーブルを繋ぐだけで自動的に有線接続に切り替わります。

  </Card>
  <Card title="Web から設定を変更できる DYA Studio 対応">
    ファームウェアを書き換えることなく、キーマップの設定やトラックボールの調整、バッテリー履歴の確認がブラウザから行えます。
    <a href="https://studio.dya.cormoran.works">DYA Studio</a> は iOS からも利用できます。

  </Card>
  <Card title="電池駆動" >
    左右はそれぞれ単４電池１本ずつで動作します。

    リチウムポリマーバッテリーと比べて発火の心配が低く、安心して使用できます。

  </Card>
  <Card title="日本語配列にも対応">
    使い慣れた Row Staggered 配列の 60% キーボードです。

    同じハードウェアで英語配列と日本語配列の両方に対応しています。

    DYA Studio も日本語配列向けの表示に対応しています。

  </Card>
  <Card title="34mm トラックボール内蔵">
    マウスを持ち替えることなくカーソル操作が可能です。

    34mmの大きなトラックボールを搭載しているので、快適に操作できます。

  </Card>
  <Card title="自由なカスタマイズ">
    ZMKファームウェアを使っておりソースコードも公開しています。

    キーマップやマクロ、細かな振る舞いを自由にカスタマイズできます。

  </Card>
</CardGrid>
